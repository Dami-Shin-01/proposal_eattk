제안일: 2025년 11월 10일 제안자: 디자인온어닷 신다미

Executive Summary

프로젝트의 핵심적인 문제점과 해결 방안을 요약합니다.

⚠️ 현재 3가지 문제

  • 낮은 웹 성능: 모바일 52점, 사용자 이탈
  • 낮은 디자인 퀄리티: 비전문적 인상, 차별화 부족
  • 모바일 최적화 미흡: 불완전한 반응형, 경험 저하

✅ 3가지 솔루션

  • Next.js 기반 고성능 웹: 90+점 목표
  • 맞춤형 디자인 & F&B 메뉴: 브랜드 차별화
  • 모바일 경험 우선: 모든 디바이스 최적 경험

Analysis

데이터로 보는 문제점과 경쟁사와의 격차를 분석합니다.

📊 성능 진단: Google PageSpeed

52 모바일
71 데스크톱

평균 로딩 5.3초, 예상 이탈률 60% 이상.
이는 비즈니스에 직접적인 타격을 줍니다.

⚔️ 경쟁사 벤치마킹: 기프트럭



디자인, 애니메이션, 모바일 UX 등에서
기프트럭에 비해 현저히 낮은 점수를 보입니다.


기프트럭 수준의 완성도를
"잇테이블코리아"만의 스토리로 구현해야 합니다.

Solution

3단계 개선 전략을 통해 문제를 해결합니다.

1

기술 기반 마련

느리고 제약 많은 WordPress 대신,
빠르고 현대적인 Next.js 14를 도입,
3배 빠른 로딩 속도와
SEO 최적화를 달성합니다.

2

디자인 차별화

잇테이블코리아만의
브랜드 아이덴티티를 담은
세련된 효과, 디자인을 적용합니다.

3

전환 최적화

사용자 여정을 기반으로
전략적인 CTA를 배치하고,
신규 F&B 메뉴와 문의를
유기적으로 연결하여
전환율을 극대화합니다.

Tech Stack

WordPress 대비 압도적인 성능, 보안,
그리고 확장성을 제공하는 현대적인 기술 스택입니다.

Next.js React Tailwind CSS Framer Motion

100% 맞춤 디자인, 기프트럭 수준의 인터랙션,
